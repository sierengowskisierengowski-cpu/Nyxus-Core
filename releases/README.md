# Releases Staging

Use this folder as the single organized location for publish-ready artifacts.

- `iso/` → final NYX ISO images and their checksum/signature files.
- `kernel/` → custom kernel packages, modules, and release notes.

Keep filenames versioned and explicit (example: `nyx-2026.05.08-x86_64.iso`,
`nyx-kernel-6.12.1-1-x86_64.pkg.tar.zst`).
