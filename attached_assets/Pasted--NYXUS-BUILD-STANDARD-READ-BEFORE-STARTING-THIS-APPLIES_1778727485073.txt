
NYXUS BUILD STANDARD — READ BEFORE STARTING. THIS APPLIES TO EVERYTHING PAST, PRESENT, AND FUTURE.
This is a professional enterprise-grade operating system build. Every single feature, app, component, widget, and UI element — everything already built and everything being built now — must meet this standard completely. No exceptions, no stubs, no placeholders, no greyed out options, no toggles that do nothing, no empty pages. If it is not complete it does not get merged.
Every feature must be highly designed, fully polished, maxed out, and feel like it belongs in a system that stands next to macOS and Windows 11. Nothing basic, nothing boring, nothing generic. Every pixel intentional. Every animation smooth. Every interaction satisfying. Stay within the existing NYXUS brand identity already established in the codebase — do not change any existing colors, fonts, or brand elements. Only enhance and polish what is already there and build new things to match it exactly.

YOUR ROLE — Frontend, Visual Design, and UI Layer
You are responsible for everything the user sees and interacts with. Copilot handles the backend wiring, system binaries, and ISO staging. You handle the visual layer — making everything look and feel premium, polished, and cohesive. When you finish a UI component Copilot will wire it to the real backend functions. Your job is to make it look and behave exactly right so the wiring is straightforward.
Do not change any existing brand colors, fonts, or established design tokens. Match everything you build to what already exists in the codebase exactly.

After every phase:
Every UI component must match the existing NYXUS brand exactly before pushing. Ask yourself — does this look like it belongs on a system worth a thousand dollars? Does it match everything else already in the build? Is every animation smooth? Is every interaction satisfying? If the answer is no — fix it before pushing.
Push all visual work to the repo so Copilot can pull it and wire the backend. Include clear comments explaining what each UI element connects to so Copilot knows exactly how to wire it.
This must ship complete, polished, and premium. If it does not match the existing brand and quality level it goes back. No compromises.
