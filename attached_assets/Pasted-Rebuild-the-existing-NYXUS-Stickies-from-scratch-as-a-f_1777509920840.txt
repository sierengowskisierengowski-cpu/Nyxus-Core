- Notes are slightly tilted — random small rotation between 
  -8 and +8 degrees per note — no two notes at same angle

  curls up slightly like a real sticky note
- Notes have a subtle drop shadow — like they are physically 
  pinned above the surface
- All text on notes uses handwritten style font — looks like 
  someone actually wrote on the note by hand
- Note borders are hand drawn — slightly imperfect, sketch-like
- Pin or tape at top of each note — hand drawn pushpin or 
  tape strip holding the note up
- When you hover a note it lifts slightly — shadow grows, 
  slight scale up animation, feels physical
- When you drag a note it tilts more — feels like picking 
  it up
- Notes slightly overlap each other naturally like a real 
  desk covered in sticky notes
- Delete animation — note crumples and falls off screen
- New note animation — note flies in and sticks to surface
- All buttons and UI elements outside the notes are hand 
  drawn sketch style — no standard GTK widgets visible
- The background canvas looks like a corkboard, dark desk 
  surface, or wall — configurable
- Scrollable infinite canvas — notes can go anywhere
