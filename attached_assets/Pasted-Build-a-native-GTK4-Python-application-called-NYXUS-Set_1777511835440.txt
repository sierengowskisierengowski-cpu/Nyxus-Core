Build a native GTK4 Python application called NYXUS Settings.
This is the complete system control center for NyX.x.OS.
Replaces and extends GNOME Settings entirely.

DO NOT design colors, theme, or NyX brand styling — that will
be applied separately. Focus entirely on functionality.
Hand drawn hand written visual style throughout — same as all
other NYXUS apps. Every element looks sketched and hand drawn.

PHILOSOPHY:
One app controls everything. Every system setting, every
hardware control, every preference, every customization —
all in one place. Clean, organized, actually works.
No hunting through terminals for config files.
No memorizing commands. Everything visual and clickable.

TECH STACK:
- GTK4 Python native
- GLib for system integration
- DBus for system service control
- NetworkManager DBus API for all network management
- UPower DBus for power management
- AccountsService DBus for user management
- systemd DBus for service management
- udev for hardware detection
- subprocess for system commands
- psutil for system information
- All operations in background threads
- Settings stored at ~/.config/nyxus-settings/
- System settings written to appropriate system paths
- All changes applied immediately — no apply button needed
  except where restart is required
- Restart required indicator shown when needed
