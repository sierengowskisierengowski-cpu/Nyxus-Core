WHAT GETS SEARCHED — REAL WORKING TOOLS ONLY:

EMAIL INVESTIGATION:
Triggers automatically when @ detected in input.
Runs all simultaneously in background threads:

HaveIBeenPwned API:
- Requires free API key — prompt user to enter in settings
- Returns every breach email appeared in
- For each breach shows:
  Breach name and website
  Date of breach
  Exactly what data was exposed
  Number of accounts in breach
  Whether passwords were plain text or hashed
- Also checks paste sites via HIBP API

Holehe:
- Install via pip — holehe package
- Checks 200+ platforms for account registered 
  with that email
- Returns for each platform:
  Account exists — confirmed
  Account likely exists
  Account not found
  Platform name and direct URL if found
- Run via subprocess, parse output

DeHashed API:
- Requires API key — prompt in settings
- Searches breach database
- Returns entries containing that email
- Shows username, IP, password hash, name if present

Username and name extraction:
- Extract username portion before @ symbol
- Extract potential first/last name if format allows
- Automatically feed both into person search
- Runs person search in parallel without user input

EMAIL RESULT CARDS show:
- Platform name with icon
- Status — found, not found, likely
- Direct link to profile if found
- Breach details if breach result
- Data exposed clearly listed
- Date of breach or discovery
- Confidence level
- Source clearly labeled

USERNAME SEARCH:
Triggers when username pattern detected or 
extracted from email.

Sherlock:
- Install via pip or git clone
- Run via subprocess with username as argument
- Parse output in real time as it streams
- For each platform found:
  Platform name
  Full URL to profile
  Response time
  Status — found, not found, error
- Show results populating in real time
- Group by: found, not found, error

Additional username checks beyond Sherlock:
- Namechk style checks for platforms Sherlock misses
- Username variations automatically checked:
  username123, username_, _username, username.name
  username-name, nameusername, name_username

USERNAME RESULT CARDS show:
- Platform name
- Full clickable URL
- Account status
- Profile preview if publicly accessible
- Date account created if visible
- Confidence level

Public Government Records APIs:
All free public APIs — no special access needed:

FAA Aircraft Registration:
- Query: registry.faa.gov API
- Search by owner name
- Returns aircraft type, tail number, 
  registration date, owner address, status
- Shows if they own or have owned aircraft

FCC License Database:
- Query: data.fcc.gov API  
- Search by name
- Returns license type, callsign, frequency,
  address, grant date, expiry, status
- Amateur radio, commercial licenses etc

SEC EDGAR:
- Query: efts.sec.gov API
- Search by person name
- Returns all filings mentioning that name
- Officer roles, company affiliations,
  stock transactions requiring disclosure

FEC Campaign Finance:
- Query: api.open.fec.gov — free public API
- Search donor name
- Returns all political donations over $200
- Amount, recipient, date, employer, address
- Completely public record

USPTO Patents:
- Query: developer.uspto.gov API
- Search inventor name
- Returns all patents filed
- Patent title, number, date, description
- Co-inventors revealed

USPTO Trademarks:
- Query: tsdr.uspto.gov API
- Search owner name
- Returns all trademark registrations
- Brand names they own or have owned

OpenSanctions:
- Query: api.opensanctions.org — free API
- Check name against all sanctions lists
- UN, EU, US OFAC, and 200+ other lists
- Returns any matches with list source
- Critical for due diligence

Wayback Machine:
- Query: archive.org API
- Search for pages mentioning the name
- Returns cached pages with that name
- Show historical content found

Reddit Search:
- Query: pushshift.io API
- Search username and name in posts and comments
- Returns posts, comments, subreddits active in
- Account creation date if findable

GitHub Search:
- Query: api.github.com — free public API
- Search by name and email
- Returns repositories, commits, contributions
- Email addresses found in commit history
- Organizations member of

Pastebin and Paste Sites:
- Search psbdmp.ws API for pastes containing name
- Returns any paste mentioning the name or email

News Search:
- Google News RSS for name mentions
- Returns recent and historical news articles
- Date, source, headline, snippet

PERSON RESULT CARDS show:

- Relevant details extracted and shown clearly
- Direct link to source
- Date if available
- Confidence level

IP INVESTIGATION:
Triggers on IP address input.
Runs simultaneously:

Shodan:
- Requires API key — free tier available
- Full host report for IP
- Open ports and services running
- Software versions detected
- Known vulnerabilities
- Historical data — what was running before
- Organization and ISP

VirusTotal:
- Requires free API key
- IP reputation report
- Detection by security vendors
- Related malicious files
- URLs hosted on this IP
- Communication patterns

AbuseIPDB:
- Requires free API key
- Abuse confidence score
- Report history — who reported it and why
- Country, ISP, usage type
- Total reports count

IPinfo:
- Free tier available
- Geolocation — city, region, country
- ISP and organization
- ASN information
- Timezone
- Whether it is a VPN or proxy or Tor

WHOIS:
- python-whois library
- Full registration data
- Registrant if not privacy protected
- Registration and expiry dates
- Name servers

Reverse DNS:
- What domain names point to this IP
- Historical reverse DNS

Tor Exit Node Check:
- Check against Tor exit node list
- Returns whether IP is a known Tor exit node

IP RESULT CARDS show:
- IP reputation score prominently
- Location on inline map
- All open ports listed clearly
- ISP and organization
- Abuse history summary
- All vendor detections
