
BEHAVIOR:
- Fixed width 460px
- Height 680px
- Opens as floating popup above taskbar button
- Opens on click OR on hover after 500ms — configurable
- Closes when clicking outside panel
- Smooth slide up animation when opening
- Smooth slide down animation when closing
- Always on top when open

WEATHER TILE — top right:
- Pulls live data from NYXUS Weather app directly
  reads from ~/.config/nyxus-weather/ config and cache
- Shows: current temp large font, condition icon,
  high and low for today, feels like temp
- Click opens NYXUS Weather app full window
- Refresh pulls fresh weather data
- Hand drawn border around tile
- Semi-transparent dark background

SYSTEM TILE — below weather:
- Live system stats pulled via psutil:
  CPU usage percentage
  RAM usage — used of total
  GPU temp and usage — nvidia-smi
  Active network — upload and download speed
- Color coded — green normal yellow warning red high
- Updates every 5 seconds
- Hand drawn border

TECHNICAL:
- GTK4 Python native
- feedparser for RSS feed reading
- requests for fetching news content
- BeautifulSoup4 for parsing article previews
- psutil for system stats
- subprocess for nvidia-smi GPU data
- Pillow for image handling in news cards
- Chromium launched via subprocess
- Weather data read from NYXUS Weather cache files
  at ~/.config/nyxus-weather/
- News cache at ~/.config/nyxus-panel/cache/
- Settings at ~/.config/nyxus-panel/config.json
- News refreshed in background thread
- UI never freezes during refresh
- Graceful fallback if feed unavailable
- Waybar integration:
  Add to Waybar config as custom module
  Click waybar button triggers panel open/close
  Panel anchors above the button position
