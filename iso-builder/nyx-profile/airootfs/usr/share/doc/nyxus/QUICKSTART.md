# NYXUS · DARK MIRROR · Quickstart

Welcome to NYXUS. A short tour:

| Keybinding         | Action                              |
| ------------------ | ----------------------------------- |
| `Super`            | NYXUS Spotlight (apps + search)     |
| `Super + Enter`    | Terminal                            |
| `Super + ?`        | Show this cheat sheet               |
| `Super + L`        | Lock screen                         |
| `Super + Shift + Q`| Power menu                          |
| `Print`            | Region screenshot                   |
| `Super + V`        | Clipboard history                   |
| `Super + N`        | Toggle notifications                |

## Logs
All NYXUS apps log to `~/.cache/nyxus/<app>.log`.

## Doctor
If anything seems off, run `nyxus-doctor` from a terminal.

## Settings
Open `NYXUS Settings` from Spotlight to configure everything.
