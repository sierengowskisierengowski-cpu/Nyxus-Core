# NYXUS Theme Archive

This directory contains alternate/legacy NYXUS theme variants that are intentionally kept outside the active default build inputs.

Active build inputs continue to live at the root of `nyxus-scripts/` and are staged by `iso-builder/build-iso.sh`.

Archived variants currently stored here:

- `boot-splash/`
- `boot-splash-void/`
- `nyxus-ui-theme/`
- `wlogout-theme/`
